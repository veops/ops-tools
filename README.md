# ops-tools

运维过程中一些常用的实践和代码，将一些运维中的小工具不断加入进来，避免重复造轮子

### 独立组件

[zabbix2tsdb](zabbix2tsdb/readme.md)  将zabbix的监控数据转换为prometheus格式输出

[netTopology](netTopology)  基于SNMP协议自动发现局域网网络拓扑关系,输出拓扑数据结构

[messenger](messenger/README.md) 简单易用的消息发送服务
